## Servidor en Deno

Servidor simple en Deno con persistencia de datos en MongoDB.


Para iniciar scripts.json (cuando estes creando un nuevo proyecto):
#### `denon --init`


Para correr el script que inicia el server, en **Clase_47/**:
#### `denon start`
